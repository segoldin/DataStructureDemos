Demos of sorting, illustrating the effects of data structure and algorithm
--------------------------------------------------------------------------

binaryTree.c - Sorted, unbalanced binary tree module

binaryTree.h - Header file for public functions in the above

bubble.c - Implementation of bubble sort for integers

dracula.txt - Input file for text sorting - about 16,000 items

dumas.txt - Input file for text sorting - more than 100,000 items

intvalues1.txt - Input file for integer sorting - 20 items

intvalues2.txt - Input file for integer sorting - about 500 items

Makefile - Build demos using Linux make

newBubble.c - Bubble sort for strings using two level data structure

sortdata1.txt - Input file for text sorting - about 30 items

sortdata2.txt - Input file for text sorting - about 300 items

sortdata3.txt - Input file for text sorting - about 600 items

stringBubble.c - Bubble sort for strings using single long array of char (very inefficient)

timeFunctions.c - Utility for measuring elapsed time, comparing algorithms

timeFunctions.h - Header file for time measuring utility

treeSort.c - Main program for implementing binary-tree-based sort
